Política de Privacidade

Última atualização: 17 de novembro de 2025

Sua privacidade é fundamental para o CartasApp. Esta política explica quais dados coletamos, por que os coletamos e como os utilizamos e protegemos, em conformidade com a Lei Geral de Proteção de Dados (LGPD) do Brasil.

O responsável pelo tratamento dos dados pessoais é o CartasApp, projeto individual mantido por Diego Rosa, pessoa física, em Blumenau/SC, Brasil. Contato principal: contato@cartasapp.com.br.

1. Quais dados coletamos

Para gerar seus documentos, coletamos as informações que você insere voluntariamente nos formulários. Isso pode incluir, mas não se limita a:

  • Dados pessoais: nome completo, CPF, e-mail, telefone, cidade/UF.
  • Dados relacionados ao documento: informações de empresas, contratos, serviços, viagens, multas, bagagens etc., conforme o tipo de carta selecionado.
  • Dados de menores (no formulário de viagem): nome completo de menores de idade, data de nascimento e números de documentos (RG, Certidão ou Passaporte) informados por você.
  • Dados de pagamento: não armazenamos dados do seu cartão de crédito. O pagamento é processado inteiramente pelo Mercado Pago, que possui sua própria política de privacidade.
  • Dados de navegação: coletamos dados anônimos/agrupados sobre como você usa o site, através de ferramentas como Google Analytics (via Google Tag) e Microsoft Clarity, para melhorar sua experiência.
  • Dados técnicos de segurança e consentimento: podemos registrar IP, tipo de navegador, data/hora e um identificador de pedido para fins de segurança, prevenção de fraude e comprovação de consentimento ao uso do serviço.

2. Como usamos seus dados

Usamos seus dados estritamente para os seguintes fins:

  • Geração do documento: as informações que você preenche são enviadas às nossas funções de backend, que utilizam modelos próprios e tecnologia de inteligência artificial (atualmente, provedores como o Google Gemini) com o único propósito de gerar o texto do documento solicitado.
  • Processamento do pagamento: usamos um ID de pedido para criar sua transação de pagamento junto ao Mercado Pago e associar o pagamento ao documento gerado.
  • Armazenamento e recuperação: armazenamos o documento gerado e o ID do seu pedido em nosso banco de dados (Supabase) para que você possa recuperá-lo após o pagamento (pelo site ou pelo link de e-mail).
  • Comunicação: usamos seu e-mail (caso você o forneça) para enviar o link de recuperação do seu documento e, eventualmente, mensagens relacionadas ao próprio serviço contratado.
  • Segurança e logs: usamos dados técnicos (IP, navegador, data/hora, versão de termos) para proteger o serviço, evitar abusos e comprovar, quando necessário, que houve aceite dos Termos de Uso antes da contratação.

Não utilizamos os dados de menores de idade para fins de marketing. Eles são tratados apenas para gerar o documento de autorização de viagem solicitado pelo responsável.

3. Compartilhamento de dados

Nós não vendemos seus dados pessoais. Seus dados são compartilhados apenas com os parceiros tecnológicos essenciais para o funcionamento do serviço:

  • Google (IA e reCAPTCHA): para geração do documento (modelos de IA) e proteção do site contra abusos e spam.
  • Supabase (banco de dados): para armazenar seu documento gerado e informações técnicas de forma segura.
  • Mercado Pago (pagamentos): para processar a transação financeira.
  • Zoho Mail ou serviço similar (e-mail): para enviar o link de recuperação do documento.
  • Netlify (hospedagem): onde nosso site e funções de backend operam.

4. Retenção e exclusão de dados

Os dados dos documentos gerados são mantidos em nosso sistema por um período razoável para permitir que você recupere o arquivo após o pagamento (por exemplo, em caso de perda do link). Após esse período, os dados podem ser removidos ou anonimizados.

Você pode solicitar a exclusão antecipada dos seus dados, lembrando que isso impossibilita a recuperação futura do documento pelo sistema.

5. Seus direitos (LGPD)

Você, como titular dos dados, tem o direito de:

  • acessar os dados pessoais que mantemos sobre você;
  • corrigir dados incompletos, inexatos ou desatualizados;
  • solicitar a exclusão dos seus dados de nosso banco de dados, quando aplicável.

Para exercer seus direitos, entre em contato conosco pelos canais abaixo. Para sua segurança, podemos solicitar comprovação de identidade antes de atender à solicitação.
